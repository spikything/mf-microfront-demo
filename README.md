# mf-microfront-demo

A microfrontend demo using **SolidJS** and **Module Federation**.

### Setup

Fork/clone the repo, then in each app folder (remote, host, host-react) do:

```
npm i
```

### Running

Inside each app folder (remote, host, host-react) do:

```
npm run dev
```

### Building

Inside each app folder (remote, host, host-react) do:

```
npm run build
```
